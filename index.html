WireGuard GUI Client
for Linux

A clean, lightweight desktop app to import your .conf files and bring your WireGuard VPN tunnel up or down with a single click. Built with Python & Flet.

Get started View on GitHub

Features

Everything you need to manage a tunnel — nothing you don't.

Import configs

Pick any WireGuard .conf through a native file dialog; it's saved locally for quick access.

One-click connect

Connect and disconnect from a dropdown with live status feedback — connected or disconnected at a glance.

Secure by design

Privileged wg-quick calls run through pkexec; the GUI itself never runs as root.

Native & cross-distro

A real desktop window powered by Flet/Flutter. Runs on Fedora, Ubuntu, Arch and more.

See it in action

The whole app fits in one tidy window.

Built with

A tiny stack — one Python dependency plus standard Linux VPN tooling.

🐍 Python 3.10+ 🦋 Flet GUI framework 🔐 wireguard-tools wg-quick / wg 🛡️ polkit pkexec ⚡ asyncio async subprocess

Deploy & run

Up and running in under a minute.

1

Install system dependencies

# Fedora / RHEL
sudo dnf install wireguard-tools polkit

# Debian / Ubuntu
sudo apt install wireguard-tools policykit-1

# Arch
sudo pacman -S wireguard-tools polkit
2

Clone the repository

git clone https://github.com/your-username/wireguard-gui-client.git
cd wireguard-gui-client
3

Create a virtual environment & install

python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt   # or: pip install flet
4

Run the app

python src/main.py            # or: flet run src/main.py
5

Build a standalone bundle (optional)

flet build linux              # output in build/linux/

⚠️ Security: WireGuard .conf files contain private keys. Keep them at chmod 600, never commit them to git, and never run the whole GUI with sudo — only wg-quick is elevated via pkexec.
